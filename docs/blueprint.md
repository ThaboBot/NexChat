# **App Name**: ChronoChat

## Core Features:

- AI Message Summarization: Auto-summarize long messages using an LLM tool.
- Real-time Translation: On-the-fly message translation into multiple languages, leveraging AI.
- Self-Destructing Messages: Option to send messages that self-destruct after a set time.
- Time Capsule Messages: Set messages to be delivered at a specific date in the future.
- Crypto Transfer: Send crypto tokens directly in chat.
- Chat Polls: Implement voting polls and analyze them with AI.

## Style Guidelines:

- Primary color: Futuristic purple (#9400D3), evocative of technological sophistication.
- Background color: A muted, dark shade of purple (#1A0029).
- Accent color: Electric blue (#7DF9FF), suggesting energy and forward momentum.
- Body and headline font: 'Space Grotesk', a contemporary sans-serif to highlight the forward thinking of the app.
- Use sleek, minimalist icons to maintain a clean and modern interface.
- Employ subtle animations for message transitions and interactive elements to create a dynamic user experience.
